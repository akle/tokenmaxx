# tokenmaxx Agent Kit

This directory is the cross-agent harness for tokenmaxx.

- `skills/` contains installed and repo-specific skills.
- `agents/` contains role personas for recurring work.
- `commands/` contains thin command procedures.
- `docs/` catalogs the kit so agents can inspect what exists.
- `settings.json` holds non-secret harness defaults.

The DeepWorkPlan skill is installed at `.agents/skills/deepworkplan`.
